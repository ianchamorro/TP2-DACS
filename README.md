# TP2-DACS DOCKER

## INSTALACION
`docker compose up -d`

`node app is running in port 3000.`

## Herramientas y tecnologias
- Docker
- Node js
- Express
- Mongo
- Mongoose

## Para parar ejecucion de el contenedor
`docker-compose stop`

## Para eliminar contenedores
`docker-compose down`
